# **App Name**: Yrdly

## Core Features:

- Main Feed: Display a scrollable feed of neighborhood posts (text, images, events, and sale listings). Show engagement metrics (likes, comments) in real-time.
- Create Post: Allow users to create new posts with: Text input, Image upload, Category selection: General, Events, Items for Sale. Support tagging locations within the neighborhood.
- Comments & Likes: Enable real-time likes and threaded comments under posts. Include emoji reactions for more expressive engagement.
- Direct Messaging: One-on-one chat between users with: Real-time updates, Read receipts, Media sharing (images, links)
- Neighborhood Map: Interactive map view showing: Posts and events pinned to exact locations. Filters for categories (e.g., Events, For Sale, General). Clickable pins that open post previews.
- Settings: Profile editing (name, avatar, bio), Password & security settings, Notification preferences (push, email), Logout option

## Style Guidelines:

- Primary Color: Soft green `#A7D1AB` — evokes community and nature.
- Background Color: Light gray `#F5F5F5` — clean, modern, and easy on the eyes.
- Accent Color: Warm orange `#FFB347` — for interactive elements and calls to action, creating a friendly contrast.
- Typography: Body & headlines: 'PT Sans' — modern, approachable, and readable.
- Icons: Use simple, outlined icons for a clean and intuitive interface.
- Layout: Card-based design with rounded corners and subtle shadows for depth. Consistent spacing and padding for readability.
- Animations: Subtle transitions for: Loading states, Button clicks, Map interactions
